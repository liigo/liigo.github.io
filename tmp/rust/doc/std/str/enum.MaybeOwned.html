Enum std::str::MaybeOwned

pub enum MaybeOwned<'a> {
    Slice(&'a str),
    Owned(~str),
}

A MaybeOwned is a string that can hold either a ~str or a &str. This can be useful as an optimization when an allocation is sometimes needed but not always.

Variants

Slice

A borrowed string

Owned

An owned string
