std::macros::debug

macro_rules! debug(
    ($($arg:tt)*) => (if cfg!(not(ndebug)) { log!(4u32, $($arg)*) })
)

A convenience macro for logging at the debug log level. See std::logging for more information. about logging.

Example

debug!("x = {x}, y = {y}", x=10, y=20);
