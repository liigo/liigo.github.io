fn binary_search_elem(&self, x: &T) -> BinarySearchResult

Binary search a sorted vector for a given element.

If the value is found then Found is returned, containing the index of the matching element; if the value is not found then NotFound is returned, containing the index where a matching element could be inserted while maintaining sorted order.
