Module std::collections::ringbufExperimental
[-]  [+] [src]

This crate implements a double-ended queue with O(1) amortized inserts and removals from both ends of the container. It also has O(1) indexing like a vector. The contained elements are not required to be copyable, and the queue will be sendable if the contained type is sendable. Its interface Deque is defined in collections.
