pub trait Collection {

A trait to represent the abstract idea of a container. The only concrete knowledge known is the number of elements contained within.

Required Methods

fn len(&self) -> uint

Return the number of elements in the container

Example

fn main() { let a = [1i, 2, 3]; assert_eq!(a.len(), 3); }
let a = [1i, 2, 3];
assert_eq!(a.len(), 3);

Provided Methods

fn is_empty(&self) -> bool

Return true if the container contains no elements

Example

fn main() { let s = String::new(); assert!(s.is_empty()); }
let s = String::new();
assert!(s.is_empty());
