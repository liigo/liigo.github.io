pub enum BinarySearchResult {
    Found(uint),
    NotFound(uint),
}

The result of calling binary_search.

Found means the search succeeded, and the contained value is the index of the matching element. NotFound means the search succeeded, and the contained value is an index where a matching value could be inserted while maintaining sort order.

Variants

Found

The index of the found value.

NotFound

The index where the value should have been found.
