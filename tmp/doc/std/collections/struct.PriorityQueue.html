A priority queue implemented with a binary heap.

This will be a max-heap.

fn with_capacity(capacity: uint) -> PriorityQueue<T>

Create an empty PriorityQueue with a specific capacity. This preallocates enough memory for capacity elements, so that the PriorityQueue does not have to be reallocated until it contains at least that many values.
