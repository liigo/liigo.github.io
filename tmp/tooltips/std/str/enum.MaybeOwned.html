pub enum MaybeOwned<'a> {
    Slice(&'a str),
    Owned(String),
}

A string type that can hold either a String or a &str. This can be useful as an optimization when an allocation is sometimes needed but not always.

Variants

Slice

A borrowed string.

Owned

An owned string.
