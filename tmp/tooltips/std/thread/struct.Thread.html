fn spawn<T, F>(f: F) -> JoinGuard<T> where T: Send, F: FnOnce() -> T, F: Send

Spawn a new joinable thread, returning a JoinGuard for it.

The join guard can be used to explicitly join the child thead (via join), returning Result<T>, or it will implicitly join the child upon being dropped. To detach the child, allowing it to outlive the current thread, use detach. See the module documentation for additional details.
